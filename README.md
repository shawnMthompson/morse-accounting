# Morse Accounting

## Installation

### Prerequisites

- Node.js 18+ and npm
- A GitHub account

### Basic Setup

1. Clone this repository:

```bash
git clone https://github.com/shawnMthompson/morse-accounting.git
cd morse-accounting
```

2. Install dependencies:

```bash
npm install
```

3. Create a `.env` file at the root of the project folder and add your [web3forms key](https://web3forms.com/) for message handling

```bash
VITE_WEB3FORM_ACCESS_KEY=YOUR_KEY_HERE
```

4. Run the development server:

```bash
npm run dev
```
