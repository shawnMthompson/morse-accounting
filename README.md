# Morse Accounting

## Installation

### Prerequisites

- Node.js 18+ and npm
- A GitHub account

### Basic Setup

1. Clone this repository:

```bash
git clone https://github.com/shawnMthompson/morse-accounting.git
cd morse-accounting
```

2. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```
